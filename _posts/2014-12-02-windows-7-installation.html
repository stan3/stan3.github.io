I'm reinstalling Windows on my laptop as part of selling it. Process turns out to be relatively easy:

  1. Download correct language iso - google for DigitalRiver Windows 7 iso
  2. Format a USB drive as NTFS with bootable flag set (possible in the Ubuntu Disks utility
  3. Copy content of ISO to the drive (sudo mount -o loop ~/Downloads/X17-59186.iso /mnt)
  4. Boot laptop off the USB drive
